SellerFast 隐私政策

最后更新：2026-05-18

我们收集什么

什么都不收集。

SellerFast 浏览器扩展不收集、不存储、不上传任何个人身份信息。你添加的商品监控列表存储在你本地浏览器的存储空间中，不会发送到任何第三方服务器。

我们如何使用数据

商品价格检查需要向我们的 API 服务器发送商品 URL 和产品 ID，仅用于查询当前价格。这些请求不包含任何个人身份信息，不关联到具体用户。

AI 翻译和评价分析功能会将文本发送至 DeepSeek API 进行处理，DeepSeek 不会将 API 调用内容用于模型训练。

数据存储

你的商品监控列表和价格历史存储在浏览器本地（chrome.storage）。你可以随时在扩展中删除任何商品，删除后数据不可恢复。

第三方服务

本扩展使用 DeepSeek API 提供 AI 翻译和分析服务。请参阅 DeepSeek 隐私政策。
