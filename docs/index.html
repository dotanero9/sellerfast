Chrome / Edge 安装教程

第 1 步：下载扩展包

  1. 前往 GitHub Releases
  2. 下载 sellerfast-chrome-v0.1.0.zip
  3. 解压到任意文件夹（解压后不要删除）

第 2 步：加载扩展

  1. Chrome：地址栏输入 chrome://extensions
  2. Edge：地址栏输入 edge://extensions
  3. 打开右上角「开发者模式」开关
  4. 点击「加载已解压的扩展」
  5. 选择刚才解压的文件夹

第 3 步：启动后端（本地服务）

  1. 确保已安装 Node.js（下载）
  2. 下载后端源码：git clone https://github.com/dotanero9/sellerfast.git
  3. 进入目录：cd sellerfast/backend
  4. 安装依赖：npm install
  5. 配置 Key：编辑 .env 填入 DeepSeek API Key
  6. 启动：node src/index.js
  7. 看到 SellerFast API running on port 3000 即可

第 4 步：开始使用

  1. 打开 Amazon 任意商品页面
  2. 右下角出现「📊 Monitor with SellerFast」按钮
  3. 点击即可添加监控
  4. 点击浏览器工具栏的扩展图标查看价格列表
