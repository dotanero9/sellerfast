## 五、回复模板

### 有人问"安全吗？会封号吗？"

```
不会。我们只读取Amazon公开的商品页面，不登录你的卖家账号，
不调Seller API，不操作后台。本质和你手动打开网页看价格一样。
数据存你本地浏览器，不上传任何服务器。
```

### 有人问"和Helium 10 / Jungle Scout有什么区别？"

```
H10和JS是重型武器，选品+关键词+广告全管，一个月$50+。
SellerFast只做一件事：日常盯价+看差评+翻译，
轻、快、打开就用。而且免费版够日常用了。
适合已经上了轨道、不想每月花几百刀买工具的卖家。
```

### 有人问"什么时候收费？"

```
免费版永远可用。付费版会增加：无限监控数量、90天价格历史、
AI深度分析、全部语种翻译。定价大概$19/月。
现在是找反馈阶段，用得好可以提前锁定Pro折扣。
```
